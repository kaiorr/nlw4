## :zap: Installation
Run the commands below to install and run the app.
  ```sh
    #Clone the project
    git clone https://github.com/kaiorr/nlw4
  ```

  ```sh
    #Go to server folder
    cd nlw4

    #Install dependencies
    yarn

    #Start the server
    yarn dev:server
   ```
